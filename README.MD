# Requirements 
Project requires Microsoft Point of Service for .NET v1.14.1 (POS for .NET)

https://www.microsoft.com/en-us/download/details.aspx?id=55758

#Setup

To use the application edit the Cashdrawer and Printers OPOS name in the appconfig

#Disclaimer

The project is not meant for any production usecase and only intended to allow finding bugs that happen when OPOS Printer/Cashdrawer is used in a certain order of commands!
